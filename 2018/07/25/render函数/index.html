render函数的第一个参数(必须)

在render函数的方法中,参数必须是createElement,它是类型是function.

render函数的第一个参数可以是:

  1. String
    html标签

  2. Object
    一个含有数据选项的对象

  3. Function
    方法返回含有数据选项的对象

render函数的第二个参数

第二个参数是可选,第二个参数是数据对象(只能是Object)

  1. class
  2. style
  3. attrs:正常的html属性
  4. domProps:原生的DOM属性

render函数的第三个参数

第三个参数是可选的

  1. String:创建文本子节点
  2. Array:创建元素子节点(VNODE)
