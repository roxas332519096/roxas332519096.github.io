Axios 的好处

从浏览器中创建 XMLHttpRequests
从 node.js 创建 http 请求
支持 Promise API
拦截请求和响应
转换请求数据和响应数据
取消请求
自动转换 JSON 数据
客户端支持防御 XSRF

axios 发送 post 请求

axios 发送 get 请求

  2. 在 vue 的入口文件 main.js 里引入 axios 并使用
